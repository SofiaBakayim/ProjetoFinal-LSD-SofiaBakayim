Funcionalidade

Como identificar uma relíquia?
Para identificar uma relíquia, tira uma fotografia do objeto e carrega na app. A nossa IA irá analisar e fornecer informações sobre a sua origem, valor e história.
Posso usar a app no computador e na televisão?
Atualmente a app está otimizada para dispositivos móveis. Estamos a trabalhar numa versão para desktop que estará disponível em breve.
O que acontece se a imagem não for reconhecida?
Se a imagem não for reconhecida, pode tentar com uma fotografia mais clara ou contactar a nossa equipa para assistência personalizada.

Conta

Preciso de conta para usar a app?
Não é obrigatório ter uma conta para usar as funcionalidades básicas. No entanto, criar uma conta permite guardar o histórico das suas identificações.
Quais as vantagens de ter uma conta?
Com uma conta pode guardar o histórico das suas identificações, receber notificações sobre objetos similares e aceder a funcionalidades avançadas.
Como recupero a conta?
Pode recuperar a sua conta através do email associado ou contactando a nossa equipa de suporte.

Dados

As imagens que envio são guardadas?
As imagens são processadas para identificação mas não são guardadas permanentemente. Apenas guardamos metadados anónimos para melhorar o serviço.
A app partilha os meus dados com terceiros?
Não partilhamos os seus dados pessoais com terceiros. Todos os dados são tratados de acordo com a nossa política de privacidade.
Como são protegidos os meus dados?
Utilizamos encriptação de ponta a ponta e seguimos as melhores práticas de segurança para proteger todos os seus dados.
