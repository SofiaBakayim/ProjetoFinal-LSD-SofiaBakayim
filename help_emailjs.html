How it works

Descobre como a Chronos funciona e como podes começar a usar a nossa plataforma para identificar relíquias e objetos antigos.

Perguntas Frequentes

Como Funciona

Como posso identificar um objeto antigo?
Basta fazer upload de uma fotografia clara do objeto na nossa plataforma. O nosso sistema de IA analisará a imagem e fornecerá informações detalhadas sobre o objeto.
Que tipos de objetos posso identificar?
Identificamos uma vasta gama de objetos antigos, incluindo móveis, joias, cerâmicas, livros, documentos históricos e muito mais.
Quão precisa é a identificação?
A nossa tecnologia oferece uma precisão superior a 95% para a maioria dos objetos. Quanto melhor for a qualidade da fotografia, mais precisa será a identificação.

Problemas Técnicos

A aplicação não reconhece a minha imagem
Se a imagem não for reconhecida, pode tentar com uma fotografia mais clara ou contactar a nossa equipa para assistência personalizada.
Esqueci-me da minha password
Pode recuperar a sua conta através do email associado ou contactando a nossa equipa de suporte.
A aplicação está lenta
Tente atualizar a página ou limpar o cache do browser. Se o problema persistir, contacte-nos.

Privacidade e Segurança

Os meus dados estão seguros?
As imagens são processadas para identificação mas não são guardadas permanentemente. Apenas guardamos metadados anónimos para melhorar o serviço.
A app partilha os meus dados com terceiros?
Não partilhamos os seus dados pessoais com terceiros. Todos os dados são tratados de acordo com a nossa política de privacidade.
Como são protegidos os meus dados?
Utilizamos encriptação de ponta a ponta e seguimos as melhores práticas de segurança para proteger todos os seus dados.
